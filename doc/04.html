現在我們指定根節點選擇器是 $(".content")。那麼，我們可以透過 vs.root 追朔到我們指定的根節點選擇器，也就是 $(".content") 來做 jQuery 的任何事情。例如 vs.root.on() 或 vs.root.find() 。

這樣的概念，能規範每個模組只做根節點之下的事情，盡可能達到不越矩，方便日後的維護。 

$(".content").vmodel("my", true, function (){
    var vs = this;
    this.autoload = ['sned']; 
    this.sned = function (){
        vs.root.on("click", ".send", function (){
            //呼叫內部的 message() 方法
            vs.message("Hello World");
        });
    }
    this.message = function (text){
        vs.root.find(".message").text(text);
    }
});
    
<div class="content">
    <div class="message"></div>
    <button class="send">送出</button>
</div>
